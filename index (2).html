Standardise

I always seem to end up with files in my system which follow different naming conventions, whether it be 'foo bar.*', 'foo_bar.*' or my preferred 'foo-bar.*'. Worse again is when upper case letters are thrown into the mix. Seeing as I primarily use a BASH terminal for file operations, naming conventions are quite important to me.

BASH is case sensitive and hitting the shift key requires effort so I prefer to have all my file names lower case. This is also my reasoning behind choosing hyphens as opposed to underscores for filenames. To solve my problem, I wrote a simple Python script to change filenames in a specified directory to my preferred file type. then moved to my bin directory to allow me to call it system wide.

Here is the code:

import os import sys work_dir = sys.argv[1] files = os.listdir(work_dir) for filename in files: new_filename = filename.replace(' ', '-').replace('_', '-') os.rename( os.path.join(work_dir, filename), os.path.join(work_dir, new_filename.lower()))
